# Antonio Vazquez Mechanical Design Portfolio

An Astro and TypeScript portfolio prepared for deployment on Vercel.

## Local development

1. Install Node.js 20 or newer.
2. Run `npm install`.
3. Run `npm run dev`.

## Production build

Run `npm run build`. The static site is written to `dist/`.

## Before deployment

Review `CONTENT_NEEDED.md` for the remaining project media and role dates. If a custom domain is added later, update `site` in `astro.config.mjs`.
