# Antonio Vazquez Mechanical Design Portfolio

An Astro and TypeScript portfolio prepared for deployment on Vercel.

## Local development

1. Install Node.js 20 or newer.
2. Run `npm install`.
3. Run `npm run dev`.

## Production build

Run `npm run build`. The static site is written to `dist/`.

## Before deployment

Review `CONTENT_NEEDED.md`, add the approved resume at `public/resume/Antonio_Vazquez_Resume.pdf`, replace `site` in `astro.config.mjs` with the final Vercel or custom-domain URL, and connect Antonio's public email address in the contact section.
